このポリシーについて

B-Sync は、BUENOFE CO SHIELD 一酸化炭素検知器を含む Brizl および Buenofe デバイス向けのスマート IoT 制御アプリです。本プライバシーポリシーでは、アプリご利用時にお客様の情報をどのように取り扱うかを説明します。

収集する情報

アプリは以下の情報を収集する場合があります：

  • アカウント認証情報（電話番号、お名前、OTP 認証）
  • デバイス登録データ（デバイス名、部屋割り当て）
  • デバイスのペアリングに使用する BLE（Bluetooth Low Energy）接続識別子
  • 接続デバイスのセンサー値（CO 濃度、温度、湿度）
  • アラーム配信のためのプッシュ通知トークン
  • アプリから送信されたカスタマーサポートの問い合わせ内容

情報の利用方法

収集した情報は以下の目的のみに使用されます：

  • ユーザー認証とアカウント管理
  • 登録済み IoT デバイスの接続・制御
  • CO・温度・湿度のリアルタイムアラート配信
  • アプリの信頼性およびパフォーマンスの改善
  • カスタマーサポートの提供

データの共有

B-Sync はお客様の個人データを第三者に販売・貸与・取引することはありません。センサーデータとデバイスコマンドはアプリ機能のためにバックエンドサーバー（AWS ソウルリージョン）にのみ送信されます。プッシュ通知には Firebase Cloud Messaging（Google）を使用します。広告や分析の第三者とデータを共有することはありません。

データセキュリティ

すべての API 通信は HTTPS を使用します。機密情報はデバイスレベルのセキュアストレージ（iOS Keychain / Android Keystore）に保存されます。BLE 通信はデバイスメーカー独自のプロトコルを使用します。

データ保存期間

アカウントデータはアカウントが有効な間保持されます。下記の連絡先にお問い合わせいただくことで、アカウントと関連データの削除をご依頼いただけます。

お子様のプライバシー

B-Sync は 13 歳未満のお子様を対象としていません。お子様から意図的に個人情報を収集することはありません。
