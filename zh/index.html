关于本政策

B-Sync 是专为 Brizl 和 Buenofe 设备设计的智能 IoT 控制应用程序，包括 BUENOFE CO SHIELD 一氧化碳探测器。本隐私政策说明我们在您使用该应用时如何处理您的信息。

收集的信息

应用程序可能收集以下信息：

  • 账户凭据（手机号码、姓名、OTP 验证）
  • 设备注册数据（设备名称、房间分配）
  • 用于设备配对的 BLE（低功耗蓝牙）连接标识符
  • 已连接设备的传感器数据（CO 浓度、温度、湿度）
  • 用于警报推送的通知令牌
  • 通过应用提交的客户支持询问内容

信息使用方式

收集的信息仅用于：

  • 用户身份验证和账户管理
  • 连接和控制已注册的 IoT 设备
  • 实时 CO、温度和湿度警报推送
  • 提升应用稳定性和性能
  • 提供客户支持

数据共享

B-Sync 不会向第三方出售、出租或交易您的个人数据。传感器数据和设备命令仅传输至我们的后端服务器（AWS 首尔区域）以实现应用功能。推送通知使用 Firebase Cloud Messaging（Google）。不与广告或分析第三方共享数据。

数据安全

所有 API 通信均使用 HTTPS。敏感凭据使用设备级安全存储（iOS Keychain / Android Keystore）保存。BLE 通信使用设备制造商的专有协议。

数据保留

账户数据在账户活跃期间保留。您可通过联系我们请求删除账户及相关数据。

儿童隐私

B-Sync 不面向 13 岁以下儿童。我们不会故意收集儿童的个人数据。
