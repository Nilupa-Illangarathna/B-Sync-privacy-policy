정책 소개

B-Sync는 브리즐(Brizl) 및 부앤오프(Buenofe) 기기를 위한 스마트 IoT 제어 애플리케이션으로, BUENOFE CO SHIELD 일산화탄소 경보기를 포함합니다. 본 개인정보처리방침은 앱 이용 시 귀하의 정보를 처리하는 방식을 설명합니다.

수집하는 정보

애플리케이션은 다음 정보를 수집할 수 있습니다:

  • 계정 인증 정보 (휴대폰 번호, 이름, OTP 인증)
  • 기기 등록 데이터 (기기 이름, 방 배정)
  • 기기 페어링을 위한 BLE(블루투스 저에너지) 연결 식별자
  • 연결된 기기의 센서 측정값 (CO 농도, 온도, 습도)
  • 알람 전달을 위한 푸시 알림 토큰
  • 앱을 통해 제출된 고객 지원 문의 내용

정보 활용 방법

수집된 정보는 다음 목적으로만 사용됩니다:

  • 사용자 인증 및 계정 관리
  • 등록된 IoT 기기 연결 및 제어
  • CO, 온도, 습도의 실시간 알림 전달
  • 앱 안정성 및 성능 개선
  • 고객 지원 제공

데이터 공유

B-Sync는 귀하의 개인정보를 제3자에게 판매, 임대 또는 거래하지 않습니다. 센서 데이터 및 기기 명령은 앱 기능을 위해 당사 백엔드 서버(AWS 서울 리전)로만 전송됩니다. 푸시 알림에는 Firebase Cloud Messaging(Google)이 사용됩니다. 광고 또는 분석 서비스와 데이터를 공유하지 않습니다.

데이터 보안

모든 API 통신은 HTTPS를 사용합니다. 민감한 자격 증명은 기기 수준의 보안 저장소(iOS 키체인 / Android 키스토어)에 저장됩니다. BLE 통신은 기기 제조사의 독점 프로토콜을 사용합니다.

데이터 보존

계정 데이터는 계정이 활성화된 동안 보존됩니다. 아래 연락처로 문의하시면 계정 및 관련 데이터 삭제를 요청하실 수 있습니다.

아동 개인정보

B-Sync는 만 13세 미만 아동을 대상으로 하지 않습니다. 아동으로부터 개인정보를 의도적으로 수집하지 않습니다.
